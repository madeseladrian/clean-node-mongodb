# Tratamento de Dados 

> ## DbAddAccount
1. ✅ Chamar o **Hasher** com os valores corretos
2. ✅ Retorna um erro se **Hasher** falhar
3. ✅ Chamar o **AddAccountRepository** com os valores corretos
4. ✅ Retorna um erro se **AddAccountRepository** falhar
5. ✅ Retorna um **account** caso a conta seja criada com sucesso
